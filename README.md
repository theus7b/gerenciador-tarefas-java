# Gerenciador de Tarefas

Este é um projeto simples de Gerenciador de Tarefas em Java. O programa permite:

- Adicionar tarefas  
- Listar tarefas  
- Marcar tarefas como concluídas  
- Remover tarefas  

# Tecnologias
- Java
- IntelliJ IDEA
- Git/GitHub
